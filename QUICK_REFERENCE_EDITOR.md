# 🚀 Quick Reference - Dashboard & Page Editor

## ⚡ 5-Minute Overview

### Dashboard
- View all your pages
- Create new pages
- Edit, publish, duplicate, delete pages
- See page status and creation date

### Page Editor
- Edit page sections (hero, features, gallery, contact)
- Live preview on right side
- Change theme
- Preview on different devices
- Save and publish

---

## 🎯 Common Tasks

### Create a Page
```
1. Click "+ Create New Page"
2. Enter title
3. Click "Create Page"
4. Redirected to editor
```

### Edit a Page
```
1. Click "Edit" on page card
2. Edit sections on left
3. See preview update on right
4. Click "Save"
```

### Publish a Page
```
1. Click "Publish" button
2. Status changes to "Published"
3. Page goes live
```

### Change Theme
```
1. Select theme from dropdown
2. See preview update instantly
3. Click "Save"
```

### Preview on Mobile
```
1. Click "Mobile" button
2. Preview width changes to 375px
3. See mobile layout
```

## 📝 Section Editors

### Hero Section
- Title
- Subtitle
- Button text
- Button URL

### Features Section
- Add/remove features
- Edit title and description
- Supports 3-6 features

### Gallery Section
- Add/remove images
- Enter image URLs
- Supports 3-8 images

### Contact Section
- Toggle enable/disable
- Toggle individual fields

---

## 💾 Save & Publish

### Save
- Click "Save" button
- Changes saved to database
- See "Saved" indicator

### Publish
- Click "Publish" button
- Page goes live
- Status changes to "Published"

### Unpublish
- Click "Unpublish" button
- Page hidden from public
- Status changes to "Draft"

## 🐛 Troubleshooting

### Changes Not Saving
- Check internet connection
- Check browser console for errors
- Try clicking "Save" again

### Preview Not Updating
- Refresh page
- Check browser console
- Try changing theme

### Page Not Loading
- Check page ID in URL
- Check authentication
- Try going back to dashboard
